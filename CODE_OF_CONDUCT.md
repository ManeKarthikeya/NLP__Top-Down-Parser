# Code of Conduct

## Our Pledge

We are committed to making participation in this project a welcoming and harassment-free experience for everyone.

## Expected Behavior

* Be respectful and professional.
* Welcome constructive feedback.
* Help create a positive learning environment.
* Respect differing viewpoints and experiences.

## Unacceptable Behavior

* Harassment or discrimination.
* Personal attacks or insults.
* Trolling or disruptive behavior.
* Publishing private information without consent.

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this Code of Conduct.

By participating in this project, you agree to follow these guidelines.
